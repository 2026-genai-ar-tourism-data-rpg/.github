# 시나리오 MVP 예시 — 「종로, 잊혀진 글씨의 비밀」

> **손으로 짠 완성형 시나리오 1개** (LLM 호출 없음). 프롬프트 엔지니어링·시나리오 로직 설계의 *정답지/템플릿*.
> 담은 것: NPC 대사 · 질의 분기 · 퀴즈(방탈출식) · 예산/상권 쿠폰 · 역사 인물 NPC · 사이드 퀘스트 · 연계(7-C).
> 작성: kys / 2026-06-19 · 관련: 기획 7-B(퀘스트 타입)·8-D(분기 대화)·7-C(연계)

## 0. 시나리오 메타

| 항목 | 값 |
|---|---|
| 제목 | 종로, 잊혀진 글씨의 비밀 |
| 지역 | 서울 종로 |
| 컨셉 | 세종대왕이 남긴 **글씨 조각(기억석)**이 종로 곳곳에 흩어짐 → 모아서 광화문에서 복원 |
| 입력 | 시작=안국역 · 끝=광화문 · 도보 · **예산 20,000원** · 2시간 · 혼자 |
| 종료조건 | **글씨 조각 4개 수집 → 광화문 세종 도깨비 앞에서 복원 = 엔딩** (대화 소진 아님, 8-D) |
| 노드 | 운현궁 → 익선동 한옥카페 → 인사동 → 🏁 광화문(세종) (+사이드: 이순신) |

## 1. 메커니즘 요약 (이 시나리오가 쓰는 것)

- **퀘스트 타입(7-B)**: 노드마다 1~2개 — `DIALOGUE`(분기 대화) · `FIND_OBJECT`(AR 수집) · `HUNT`(AR 요괴 처치) · `QUIZ`(방탈출식) · `PHOTO`(사진).
- **목표 카운터(RPG식)**: 노드 목표를 진행바로 — **아이템 획득 N/M** · **먹그림자 처치 N/M**. 다 채워야 조각 해금.
- **방탈출식 NPC 루프**: NPC가 **지령(미션)** 을 주고, 막히면 **단계별 힌트**(힌트1→힌트2→정답근접)를 흘림. "지령→탐색→막힘→힌트→해결"의 방탈출 루프.
- **분기 처리**: 선택지 → 결과(대사·플래그·보상·친밀도)가 갈리되 **항상 "조각 획득"으로 수렴**(branch & bottleneck, 8-D).
- **퀴즈**: 정답 → 보너스(경험치·쿠폰↑) / 오답 → 힌트 후 재시도(진행은 항상 가능, 데드락 ❌).
- **예산·상권 쿠폰**: 식음 노드(카페·식당) 예상지출 합 ≤ 예산. 노드 완료 보상에 **다음 상권 쿠폰** → 다음 식음 노드에서 차감.
- **연계(7-C)**: 앞 노드 `grants`(조각·단서·플래그)를 다음 노드가 `requires`로 읽고 NPC가 인지.

> **방탈출 루프 한 장**
> ```
> NPC 지령: "처마 그늘에 번진 먹그림자 5마리를 쫓고, 그 아래 글씨를 찾아라"
>   → AR 탐색 (먹그림자 처치 0/5 → 5/5)         ← HUNT 카운터
>   → [막힘] NPC 힌트1: "그늘은 해가 드는 반대편이니라"
>   → [더 막힘] 힌트2: "이로당 처마를 보거라"
>   → 글씨 조각 발견·수집 (아이템 1/1)            ← FIND_OBJECT
>   → 조각 해금 → 다음
> ```
> 지령·힌트 *문구*는 LLM이 grounding+페르소나로 생성, 카운터·해금·힌트 단계는 **코드가 통제**(8-D 구조).

### 1-1. 퀘스트 패턴 카탈로그 (노드마다 1~2개 조합)
> 노드별로 아래 패턴을 골라 끼운다. 타입을 섞을수록 단조롭지 않음.

| # | 패턴 | 흐름 | 성공 조건 |
|---|---|---|---|
| P1 | **대화→수집** | 도깨비 분기 대화(힌트) → AR 조각 탭 | 장소인증 + 조각 1 |
| P2 | **사냥→수집**(HUNT) | 먹그림자 N마리 처치 → 조각 등장 | 처치 N/N + 조각 1 |
| P3 | **퀴즈→개봉**(방탈출) | 퀴즈 정답 → 잠긴 상자 개봉 → 조각 | 퀴즈 정답 + 조각 1 |
| **P4** | **사진→발자국 추적→파편** ⭐ | ① 건축요소 촬영 인증 → ② 먹물 발자국 등장 → 따라가면 파편 → 탭 수집 | 장소인증 + **사진 미션** + AR 파편 1 |
| P5 | **사진 미션**(사이드) | 지정 대상 촬영 인증 | 사진 1 |
| P6 | **수집 누적**(COLLECT) | 흩어진 단서 조각 N/N 모으기 | 아이템 N/N |

→ 한 코스에 P1~P6를 섞으면 "대화→사냥→퀴즈→사진추적"으로 매 노드가 다르게 느껴짐.

## 2. 노드 상세

### 노드 1 — 운현궁 🏛 (관광지)
- **NPC**: **먹 도깨비** (장소 도깨비 / 모티프=붓·먹 / 말투="~니라, 허허")
- **퀘스트 타입**: DIALOGUE(분기) + FIND_OBJECT(AR) + QUIZ
- **requires**: 없음(시작 노드) · **grants**: 글씨조각1, 단서 "申時(신시)"

**① 등장 대사**
> *먹 도깨비*: "허허, 운현궁에 발을 들였구나. 흥선대원군이 살던 이 사저에… 어느 날 세종 임금의 글씨 한 조각이 먹물 속으로 숨어버렸느니라. 자네, 글을 아끼는 자인가?"

**② 질의 → 분기**

| 선택지 | 결과(대사) | 효과(플래그/보상) |
|---|---|---|
| A. "세종대왕의 글씨라니, 무슨 일이오?" | "훈민정음이 흩어졌느니, 백성의 글이 잠들었지." | flag `호기심`+ , 친밀도+1 |
| B. "보상은 무엇이오?" | "허허, 셈부터 빠르구나. 글씨엔 옛 기록의 힘이 깃들었지." | flag `실리`+ (이후 쿠폰 보상 +100원) |
| C. "그냥 빨리 찾겠소." | "성격 급한 게로구나. 그럼 따라오너라." | (변화 없음, 바로 다음) |

> ⮑ **분기 결과가 달라도 다음은 모두 "퀴즈 → AR"로 수렴**(bottleneck).

**③ 퀴즈 (방탈출식)**
> *먹 도깨비*: "글씨를 찾으려면 이 집의 주인을 알아야 하느니. **운현궁은 누구의 집이었더냐?**"
> - 1) 세종대왕  2) **흥선대원군** ✅  3) 정조
- **정답(2)** → "옳거니! 안목이 있구나." → 경험치 +30, **쿠폰 +200원**
- **오답** → "허허, 다시 보거라. 고종의 아버지니라." → 힌트 후 재시도 (페널티 없음, 진행 가능)

**④ 사진 미션 → AR 발자국 추적 → 파편 (패턴 P4)** ⭐
> *먹 도깨비 지령*: "글씨를 찾으려거든, 먼저 이 집의 모습을 마음에 담거라. **대문이나 처마**를 화면에 들이거라 — 그 다음 길이 열리느니."

| 단계 | 내용 | UI / 성공 판정 |
|---|---|---|
| ① 사진 미션(PHOTO) | **운현궁 건축요소**(대문·마당·전통건물 외관) 화면에 담아 촬영 | 대상 인식 → 사진 인증 ✅ |
| ② 발자국 등장 | 인증 후 카메라에 **검은 먹물 발자국**이 떠오름 | "발자국을 따라가거라" |
| ③ 발자국 추적(FIND) | 발자국 따라가면 끝에 **작은 글씨 파편** 출현 | 발자국 3개 → 파편 |
| [막힘] 힌트 | "발자국은 해 지는 쪽으로 번졌느니" → "이로당 처마 아래니라" | 단계 힌트(방탈출) |
| ④ 파편 수집 | 파편 탭 | **글씨조각 1/1** |

**성공 조건**: 장소 인증 ✅ + 사진 미션 ✅ + AR 파편 1개 ✅ → 노드 완료
> (사진 인식·발자국·파편 = AR 단계. 실제 카메라/3D는 정찬희. 인식 못 하면 "다시 담아 보거라" 폴백)

**⑤ 보상**
- 💎 글씨조각1 · ✨경험치+50 · 🔖단서 "申時" (다음 노드 연계) · 🎟 익선동 카페 **500원 쿠폰**

### 노드 2 — 익선동 한옥 카페 ☕ (식음 · 예산)
- **NPC**: **한옥 도깨비** (장소 도깨비 / 모티프=기와·차)
- **퀘스트 타입**: DIALOGUE + (주문 인증) + QUIZ
- **requires**: 단서 "申時" (있으면 특별 대사) · **grants**: 글씨조각2, 단서 "ㄱ"
- **예상 지출**: 커피 5,000원 − **쿠폰 500원 = 4,500원** (예산 20,000 중)

**① 등장 대사 (연계 인지)**
> *한옥 도깨비*: "허허, 운현궁에서 **申時 단서**를 얻어 왔구나! 그 시각, 이 골목 가마솥에 글씨 하나가 떨어졌지. **차 한 잔** 시키고 천천히 둘러보거라."
> *(주문 인증 = 카페에서 메뉴 주문 후 영수증/위치 인증 → 쿠폰 차감)*

**② 질의 → 분기**

| 선택지 | 결과 | 효과 |
|---|---|---|
| A. "이 골목은 왜 한옥이 많소?" | "1930년대 한옥 단지였느니, 도시 속 옛 시간이지." | 친밀도+1, flag `한옥통` |
| B. "추천 메뉴가 있소?" | "쌍화차에 글씨의 온기가 도느니." | (가벼운 분위기, 변화 없음) |

**③ 퀴즈**
> "이 골목 이름 **익선동**의 '익'은 무엇을 뜻하겠느냐?"
> - 1) 날개  2) **더할 익(益)** ✅  3) 물
- 정답 → 쿠폰 +300원 / 오답 → 힌트

**④ 보상**
- 💎 글씨조각2 · 🔖단서 "ㄱ"(자음 조각) · 🎟 인사동 식당 **1,000원 쿠폰**

- **NPC**: **붓장수 도깨비**
- **퀘스트 타입**: PHOTO(사진 미션) + QUIZ

**② 사진 미션 (PHOTO)**

**④ 보상**

**③ 복원 & 보상**
- 4조각 합체 → **「종로 글씨 기억석」 복원** → 엔딩 컷신
- 보상: 칭호 "종로의 글지기" · 세종 도깨비 도감 등록 · 경험치 +200 · (굿엔딩 시 희귀 유물)
- **다음 지역(북촌·경복궁 내부) 해금**

## 3. 사이드 퀘스트 (선택) — 광화문 이순신 장군 ⚔️
- **NPC**: **이순신 장군** (역사 인물 / 말투=과묵·결연)
- **트리거**: 광화문 도착 시 메인과 별개로 등장
- **퀘스트**: PHOTO — "거북선의 위용을 화면에 담으라"(동상 촬영) + QUIZ("학익진은 무슨 모양인가?" → 학이 날개 편 모양)
- **보상**: 유물 "충무공의 나침반"(조각 감지 범위↑) · 도감 등록
- **연계**: 이 나침반이 있으면 다음 지역 AR 탐색 힌트 범위↑ (7-C 전역 유물)

## 4. 예산·상권 쿠폰 흐름 (예산 20,000원)
```
운현궁(무료) ── 완료 → 카페 500원 쿠폰
익선동 카페  5,000 − 500 = 4,500원 지출 ── 완료 → 식당 1,000원 쿠폰
인사동 식당  8,000 − 1,000 = 7,000원 지출
─────────────────────────────────
총 지출 11,500원  ≤ 예산 20,000원 ✅ (여유 8,500)
```
- **규칙**: 식음 노드 예상지출 합 − 누적 쿠폰 ≤ 예산. 퀴즈 정답·실리 플래그로 쿠폰↑ → 실질 지출↓.
- 쿠폰은 **상권 제휴**(실 할인) 또는 게임 내 재화로 시작 → 제휴는 발전.

## 6. 이게 설계에 주는 것
1. **프롬프트 엔지니어(박준형)**: `intro`/`choices`/`퀴즈 문구` 생성 프롬프트의 입력·출력 형태 + 톤(도깨비/역사인물) 정답지.
2. **시나리오 로직(김예슬)**: 분기(플래그)·퀴즈 채점·예산/쿠폰·연계(grants/requires)·종료조건의 상태머신 명세.
3. **노드 데이터(이지선)**: NPC 페르소나·퀴즈·grounding 텍스트가 노드에 뭐가 필요한지.
4. **재미 검증**: 대화→퀴즈→AR→보상→연계가 한 노드에 어떻게 엮이는지 실제 텍스트로 확인.
