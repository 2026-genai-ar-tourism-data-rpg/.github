# .github (조직 프로필 레포)

> 조직 공통 문서를 담는 특수 레포. 조직 메인 페이지에 노출되는 프로필 README와 공통 템플릿을 관리합니다.

## Clone

```bash
git clone https://github.com/2026-genai-ar-tourism-data-rpg/.github.git
cd .github
```

## 내용

- `profile/README.md` — 조직 메인에 노출되는 서비스 소개·아키텍처 다이어그램
- 공통 이슈/PR 템플릿 (`ISSUE_TEMPLATE/`, `PULL_REQUEST_TEMPLATE.md`)
- 브랜치 전략·커밋 컨벤션 문서
- 제안서 PDF 보관 (`report/`)
- 레포 가이드 (`repos/`)

> GitHub은 `.github` 레포의 `profile/README.md`를 조직 홈 화면에 자동으로 표시합니다.
